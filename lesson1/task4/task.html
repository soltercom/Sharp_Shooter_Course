Description

There are a couple of ideas on how to improve our program.

First, in the real world, objects move due to the laws of physics. Generally, it is a movement with acceleration, not linear. In the physical world, objects don’t start or stop instantaneously. Instead, they take time to speed up and slow down. The following figure shows the difference between the characteristics of motion in our program and movement with acceleration.

So far in our program, when we move the Gun Sight, for instance, to the right, we increase the horizontal coordinate once (x += 10) and update the screen. But to simulate accelerated motion, we have to change this algorithm. We should create a sequence of changes to the horizontal variable, for instance, (1, 2, 4, 2, 1). For each element of this collection, we should do the following operations:

  • x += 10;
  • update screen;
I hope the idea is clear.

Second, in real life, it is not so easy to hit a target. Hands are shaking, the wind blows, gun defective. It is all the work of some forces. Under Newton’s second law, force produces motion with acceleration. That is, you can apply the algorithm discussed above.

We have considered one of a million options that we can use, so it is up to you to choose.
